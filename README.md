## üìå Introduction
Welcome to **Panaversity Assistant (AI Employee)**. This is not just a chatbot; it is a **fully autonomous Digital FTE (Full-Time Equivalent)** designed to work alongside you 24/7. It adheres to the **Watcher-Brain-Vault** architecture, ensuring privacy, persistence (Ralph Wiggum Loop), and multi-channel capability (WhatsApp, Email, LinkedIn, Odoo).

## üöÄ Key Features (Judges Overview)

### 1. **Autonomous "Senses" (Watchers)**
- **Gmail Watcher**: Monitors inbox for leads, invoices, and academic updates.
- **WhatsApp Watcher**: Reads messages using Computer Vision (Playwright) to detect urgency.
- **System**: Zero-touch operation. Triggers the brain only when "Needs Action" items are found.

### 2. **The "Brain" (Reasoning Engine)**
- Powered by **Gemini 2.5 Flash / 1.5 Pro**.
- **Ralph Wiggum Loop**: It doesn't give up. It retries tasks until they are moved to `/Done`.
- **Plan Generation**: Creates `PLAN_xyz.md` files before executing complex actions.

### 3. **Sales Command Center (Dashboard)**
- A premium, glassmorphism UI built with **Next.js 15 & Tailwind**.
- Real-time visualization of Leads, Revenue, and Tasks.
- **CEO Briefing**: Auto-generates weekly summaries of business health.

### 4. **Multi-Tool Integration (MCP)**
- **Odoo CRM**: Auto-create leads and search customers.
- **LinkedIn**: Auto-post updates and track engagement.
- **Gmail & WhatsApp**: Two-way communication.
